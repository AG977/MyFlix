MyFlix!

Goals:

To build the server-side component of a movies web application. The web application will provide users with access to information about different movies, directors, and genres.

User Goals:

Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request body data format Response body data format
Get a list of Movies /movies GET None A JSON object holding data about all the Movies
Get a list of Users /users GET None A JSON object holding data about all the Users
Get data about a single Movie by title /movies/:title GET None A JSON object holding data about a single Movie, cntaining the Title, Description, Genre, Directors, and an image. For example: { Title:"Madagascar", Description:"A group of animals who have spent all their life in a New York zoo end up in the jungles of Madagascar, and must adjust to living in the wild.", Directors: { Name:"Tom McGrath", Bio:"", BirthYear:"" } Genre: { Name:"Family", Description:"" } }
Get data about a genre /movies/genre/:genreName GET None A JSON object holding data about the genre. For example: { Genre: { Name:"Family", Description:"" } }
Get data about a director /movies/directors/:directorName GET None A JSON object holding data about a single Director, containg: Name, Bio, Date of Birth/Death. For example: { Name:"Tom McGrath", Bio:"is an American voice actor, animator, screenwriter, and film director." Birth Year:"August 7, 1964" Death Year:"/" }
Get a User by Username /users/:Username GET None A JSON object holding data about a single User, cntaining his/hers, Username, Password, Email, Date of Birth, and favoritemovies list. For example: { "Username": "Moka", "Password": "Myflix123", "Email": "123@google.com", "DateOfBirth": "01/01/2020", "FavoriteMovies": [""] }
Add a user /users POST A Jason object holding data about the user to add, structured like: { Username:"", Password:"", Email:"", DateOfBirth":"", favouriteMovies: [] } A Jason object holding data about the user that was added, including an ID: { ID:"", Username:"", Password:"", Email:"", DateOfBirth":"", favouriteMovies: [] }
Allow a user to update their information /users/:Username PUT None A text message indicating the users: Username, Date of Birth, Password, email that was updated
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST A Jason object containg a list of title's about the User's Favorite Movie. Structured as follows: { favouriteMovies: ["", ""] } A Jason object holding data about the user, adding the movie to the list of favorites by MovieID: { ID:"", Username:"", Password:"", Email:"", DateOfBirth":"", favouriteMovies: [""] }
Allow users to remove a movie from their list of favorites /users/:Username/movies/:MovieID DELETE None A Jason object holding data about the user, removing the movie of the list of favorites by MovieID: { ID:"", Username:"", Password:"", Email:"", DateOfBirth":"", favouriteMovies: [] }
Allow existing users to deregister /users/:Username DELETE None A text message indicating whether the user was successfully deleted
