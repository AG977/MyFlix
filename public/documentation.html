MyFlix!

Goals:

To build the server-side component of a movies web application. The web application will provide users with access to information about different movies, directors, and genres.

User Goals:

Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request body data format Response body data format
Get a list of Movies /movies GET None A JSON object holding data about all the Movies
Get data about a single Movie by title /movies/:title GET None A JSON object holding data about a single Movie, cntaining the Title, Description, Genre, Directors, and an image. For example: { Name:"Madagascar", Description:"A group of animals who have spent all their life in a New York zoo end up in the jungles of Madagascar, and must adjust to living in the wild.", Genre:"Family/Comedy", Directors:"Tom McGrath, Eric Darnell" }
Get data about a genre /movies/genre/:genreName GET None A JSON object holding data about the titles of the movies corresponding to the different genre's. For example: { Genre:"Family/Comedy", corresponding title's:"Madagascar" }
Get data about a director /movies/directors/:directorName GET None A JSON object holding data about a single Director, containg: Name, Bio, Date of Birth/Death. For example: { Name:"Tom McGrath", Bio:"is an American voice actor, animator, screenwriter, and film director." Birth Year:"August 7, 1964" Death Year:"/" }
Add a user /users POST A Jason object holding data about the user to add, structured like: { Name:"", Username:"", Date of Birth:"", e-mail:"" Password:"", favouriteMovies: [] } A Jason object holding data about the user that was added, including an ID: { ID:"", Name:"", Username:"", Date of Birth:"", e-mail:"" Password:"", favouriteMovies: [] }
Allow a user to update their information /users/:id PUT None A text message indicating the user, Username, Date of Bearth, Password, email that was updated
Allow users to add a movie to their list of favorites /users/:name/:newfavoritemovies POST A Jason object containg a list of title's about the User's Favorite Movie. Structured as follows: { Title:"", Title:"", } A text message indicating that a movie has been added to their Favorite's list
Allow users to remove a movie from their list of favorites //users/:name/:favoritemovies DELETE None A text message indicating the user whether the movie was successfully removed from its favorite list
Allow existing users to deregister /users/deregister DELETE None A text message indicating whether the user was successfully removed
